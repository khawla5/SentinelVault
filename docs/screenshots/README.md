# Screenshots

Place the interface screenshots in this folder using these exact filenames
(the main README references them):

| Filename | What to capture |
|---|---|
| `login.png` | The login / "Create your vault" screen |
| `vault.png` | The dashboard with a few saved entries + the strength meter in use |
| `settings.png` | The settings panel open, showing theme + color options |
| `arabic-rtl.png` | The app switched to العربية (shows RTL localization) |

**On macOS:** press `Cmd + Shift + 4`, drag to select the area, and the image
saves to your Desktop. Rename it and move it here.

Tip: use demo data (e.g. `demo@example.com`) rather than real accounts, since
these images are public on GitHub.
